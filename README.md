
Event Management Program README
This C# program, named Lab_7_Samsonenko_program, is an event management application designed to organize and manage various types of events. The program provides functionalities for creating events, displaying event information, searching for events, and performing other related operations.

Features
Creation of events with different types, such as ChildrensParties, BirthdayParties, ThemeParties, Anniversaries, and Weddings.
Input validation for event properties, including name, people count, and age.
Displaying event information in the console or saving it to a file.
Dynamic pricing based on event type, people count, and age.
Searching for events based on age, people count, name, or adult people count.
Deleting events based on specified criteria.
Demonstration of program behavior and static methods.
